Problem Set 1: Linear Programming Formulation

Q1.1 (True/False)

A linear programming problem can have a non-linear objective function as long as all constraints are linear.

☐ True
☐ False

Answer: False

Explanation: A linear programming problem requires BOTH the objective function AND all constraints to be linear. If the objective is non-linear, it's not an LP.

Q1.2 (MCQ)

Which of the following is NOT a standard form requirement for LP?

A) All variables must be non-negative
B) All constraints must be equalities
C) The objective must be maximization
D) Right-hand side must be non-negative

Answer: C

Explanation: Standard form can have either maximization or minimization objectives. We can convert between them using: max f(x) = -min(-f(x)).

Q1.3 (Fill in the blank)

To convert the constraint 3x₁ - 2x₂ ≥ 5 to standard form (equality), we add a ________ variable.

Answer: surplus (or slack with negative coefficient)

Explanation: For ≥ constraints, we subtract a surplus variable: 3x₁ - 2x₂ - s = 5, where s ≥ 0.

Problem Set 2: Polyhedra and Feasible Regions

Q2.1 (MCQ)

Which of the following sets is a polyhedron?

A) {(x,y) | x² + y² ≤ 1, x,y ≥ 0}
B) {(x,y) | xy ≤ 4, x,y ≥ 0}
C) {(x,y) | x + 2y ≤ 6, 3x - y ≤ 9, x,y ≥ 0}
D) {(x,y) | |x| + |y| ≤ 5}

Answer: C

Explanation: A polyhedron must have all linear constraints. Option C has only linear constraints. Options A and B have non-linear constraints. Option D has absolute values (though it CAN be reformulated as a polyhedron, in its current form it's not explicitly one).

Q2.2 (True/False)

The constraint x² - 6x + 8 ≤ 0 can define a polyhedron.

☐ True
☐ False

Answer: True

Explanation: x² - 6x + 8 ≤ 0 factors as (x-2)(x-4) ≤ 0, which gives 2 ≤ x ≤ 4. This is equivalent to two linear constraints: x ≥ 2 and x ≤ 4, which define a polyhedron (line segment).

Q2.3 (MCQ)

A bounded polyhedron in ℝⁿ with non-empty interior must have at least how many extreme points?

A) 1
B) n
C) n+1
D) 2ⁿ

Answer: C (n+1)

Explanation: A bounded polyhedron with non-empty interior in ℝⁿ must have at least n+1 extreme points (vertices). For example, a triangle in ℝ² has 3 vertices, a tetrahedron in ℝ³ has 4 vertices.

Problem Set 3: Matrix Rank and Linear Independence

Q3.1 (Fill in the blank)

If matrix A can be written as A = xy' where x ∈ ℝᵐ and y ∈ ℝⁿ (both non-zero), then rank(A) = ________.

Answer: 1

Explanation: Every row is a scalar multiple of every other row, so the matrix has rank 1.

Q3.2 (True/False)

If a 5×7 matrix has rank 3, then its columns span ℝ⁵.

☐ True
☐ False

Answer: False

Explanation: The columns span a 3-dimensional subspace of ℝ⁵, not the entire ℝ⁵. To span ℝ⁵, we would need rank 5.

Q3.3 (MCQ)

A set of m vectors in ℝⁿ (where m > n) can be linearly independent:

A) Always
B) Never
C) Only if m = n+1
D) Only in special cases

Answer: B (Never)

Explanation: At most n vectors in ℝⁿ can be linearly independent. If you have more than n vectors, they must be linearly dependent.

Problem Set 4: Extreme Points

Q4.1 (MCQ)

Consider the polyhedron P = {(x₁,x₂) | x₁ + x₂ ≤ 3, x₁,x₂ ≥ 0}. How many extreme points does it have?

A) 2
B) 3
C) 4
D) Infinitely many

Answer: B (3)

Explanation: The extreme points are: (0,0), (3,0), and (0,3). These are the vertices of a triangle.

Q4.2 (True/False)

Every point in a polyhedron can be expressed as a convex combination of its extreme points.

☐ True
☐ False

Answer: False

Explanation: This is only true for BOUNDED polyhedra. For unbounded polyhedra, you need extreme points plus extreme directions (rays).

Q4.3 (Fill in the blank)

A point x is an extreme point of polyhedron P if and only if there exist ________ linearly independent binding constraints at x.

Answer: n (where n is the dimension/number of variables)

Explanation: An extreme point in ℝⁿ must have n linearly independent binding constraints.

Problem Set 5: Basic Feasible Solutions (BFS)

Q5.1 (MCQ)

For a system Ax = b with m equations and n variables (n > m), a BFS has exactly:

A) n basic variables
B) m basic variables
C) n-m basic variables
D) It varies

Answer: B (m)

Explanation: A BFS has exactly m basic variables (corresponding to m linearly independent columns of A) and n-m non-basic variables (set to 0).

Q5.2 (True/False)

All basic solutions are feasible.

☐ True
☐ False

Answer: False

Explanation: A basic solution may have negative components, making it infeasible. Only basic solutions with all components ≥ 0 are called Basic Feasible Solutions (BFS).

Q5.3 (MCQ)

If we have 3 equations and 7 variables, how many ways can we choose a basis (before checking feasibility)?

A) 21
B) 35
C) 42
D) 210

Answer: B (35)

Explanation: We need to choose 3 columns from 7, which is C(7,3) = 7!/(3!·4!) = 35.

Q5.4 (Fill in the blank)

A BFS where one or more basic variables equal zero is called a ________ BFS.

Answer: degenerate

Explanation: Degeneracy occurs when a basic variable is zero, meaning more than n constraints are binding at that point.

Problem Set 6: Adjacent Basic Feasible Solutions

Q6.1 (True/False)

Two BFS are adjacent if they share exactly (n-1) binding constraints, where n is the number of variables.

☐ True
☐ False

Answer: True

Explanation: Adjacent BFS differ in exactly one basic variable (one enters, one leaves), so they share n-1 binding constraints.

Q6.2 (MCQ)

When moving from one BFS to an adjacent BFS, how many variables change their basic/non-basic status?

A) Exactly 1
B) Exactly 2
C) At most 2
D) It depends

Answer: B (Exactly 2)

Explanation: One variable enters the basis (becomes basic) and one variable leaves the basis (becomes non-basic). Total: 2 variables change status.

Q6.3 (Fill in the blank)

In the Edge Direction Method, when moving from BFS x to an adjacent BFS, the direction vector d must satisfy ________ = 0.

Answer: Ad (or A·d)

Explanation: To maintain the equality constraints Ax = b while moving along direction d, we need Ad = 0.

Problem Set 7: Edge Direction Method

Q7.1 (MCQ)

Starting from BFS (0,0,3,3) with constraints x₁+2x₂+x₃=3, 2x₁+x₂+x₄=3, x≥0. To move toward an adjacent BFS by increasing x₁, the direction vector d₁ has which component for x₁?

A) 0
B) 1
C) -1
D) Depends on other variables

Answer: B (1)

Explanation: When we choose to increase x₁ (the entering variable), we set d₁ = 1. The other components of d are determined by Ad = 0.

Q7.2 (True/False)

The maximum step size θ when moving along direction d from point x is determined by θ = min{xᵢ/dᵢ | dᵢ > 0}.

☐ True
☐ False

Answer: False

Explanation: The correct formula is θ = min{-xᵢ/dᵢ | dᵢ < 0}. We only consider negative components of d because those are the variables that decrease.

Q7.3 (Fill in the blank)

When using Edge Direction Method, the variable that becomes zero first (determines θ) is called the ________ variable.

Answer: leaving (or departing)

Explanation: The leaving variable exits the basis and becomes non-basic (set to 0).

Problem Set 8: Duality - Formulation

Q8.1 (MCQ)

For the primal: max c'x s.t. Ax ≤ b, x ≥ 0, the dual is:

A) min b'y s.t. A'y ≥ c, y ≥ 0
B) max b'y s.t. A'y ≤ c, y ≥ 0
C) min b'y s.t. A'y ≤ c, y ≥ 0
D) max b'y s.t. A'y ≥ c, y ≥ 0

Answer: A

Explanation: Primal max → Dual min. Primal ≤ constraints with x ≥ 0 → Dual y ≥ 0 with ≥ constraints.

Q8.2 (True/False)

If the primal has 5 constraints and 8 variables, the dual has 8 constraints and 5 variables.

☐ True
☐ False

Answer: True

Explanation: Primal constraints become dual variables, and primal variables become dual constraints. So roles are swapped.

Q8.3 (Fill in the blank)

The dual of the dual problem is the ________ problem.

Answer: primal

Explanation: Taking the dual twice returns you to the original problem.

Problem Set 9: Weak Duality Theorem

Q9.1 (True/False)

For a maximization primal, weak duality states that any feasible primal objective ≤ any feasible dual objective.

☐ True
☐ False

Answer: True

Explanation: This is the fundamental weak duality theorem: c'x ≤ b'y for all feasible x and y.

Q9.2 (MCQ)

If the primal is unbounded, what can we say about the dual?

A) The dual is also unbounded
B) The dual is infeasible
C) The dual has finite optimal value
D) Cannot determine

Answer: B

Explanation: If the primal is unbounded, the dual must be infeasible (by weak duality). If the dual were feasible, it would bound the primal.

Q9.3 (MCQ)

You have a feasible primal solution with objective value 50 and a feasible dual solution with objective value 45 (for max primal). What can you conclude?

A) The primal solution is optimal
B) The dual solution is optimal
C) There's an error in calculation
D) Both could be improved

Answer: C

Explanation: By weak duality, primal objective ≤ dual objective. Having 50 > 45 violates this, so there must be a calculation error.

Problem Set 10: Strong Duality Theorem

Q10.1 (True/False)

Strong duality states that if both primal and dual have feasible solutions, they both have optimal solutions with equal objective values.

☐ True
☐ False

Answer: True

Explanation: This is the strong duality theorem for linear programming. At optimality, primal objective = dual objective.

Q10.2 (MCQ)

You find feasible solutions for both primal (max) and dual (min) with objectives 100 and 100 respectively. What can you conclude?

A) Neither is optimal
B) Both are optimal
C) Only primal is optimal
D) Cannot determine without more info

Answer: B

Explanation: When feasible solutions have equal objectives, by weak duality both must be optimal (there's no gap to close).

Q10.3 (Fill in the blank)

The difference between the dual objective and primal objective is called the ________ gap.

Answer: duality

Explanation: The duality gap = dual objective - primal objective. At optimality, this gap is zero (by strong duality).

Problem Set 11: Complementary Slackness Conditions

Q11.1 (MCQ)

At optimality, if primal constraint i has slack (not binding), then:

A) yᵢ* = 0
B) yᵢ* > 0
C) All dual variables = 0
D) xᵢ* = 0

Answer: A

Explanation: Complementary slackness: If a primal constraint has slack, the corresponding dual variable must be zero.

Q11.2 (True/False)

If xⱼ* > 0 in the optimal primal solution, then the corresponding dual constraint j must be binding.

☐ True
☐ False

Answer: True

Explanation: This is the other direction of complementary slackness: positive primal variable → binding dual constraint.

Q11.3 (MCQ)

Given optimal primal x* = (5, 0, 3) for problem with 3 variables. How many dual constraints must be binding?

A) At least 1
B) At least 2
C) All 3
D) Cannot determine

Answer: B (At least 2)

Explanation: Since x₁* = 5 > 0 and x₃* = 3 > 0, dual constraints 1 and 3 must be binding. Constraint 2 may or may not be binding (x₂* = 0).

Q11.4 (Fill in the blank)

The complementary slackness conditions state that at optimality, either a constraint is binding OR its corresponding variable is ________, but not both strictly positive/active.

Answer: zero

Explanation: For each pair (constraint, variable), at least one must be zero/slack at optimality.

Problem Set 12: Shadow Prices (Dual Variables)

Q12.1 (MCQ)

A shadow price (dual variable) of 8 for a resource constraint means:

A) The resource costs $8
B) We should buy 8 more units
C) Increasing the resource by 1 unit increases profit by $8 (within range)
D) The optimal solution uses 8 units

Answer: C

Explanation: Shadow price = marginal value of the resource. It tells you how much the objective improves per unit increase (within the allowable range).

Q12.2 (True/False)

If a constraint has a shadow price of 0, it means the constraint is not binding.

☐ True
☐ False

Answer: True

Explanation: Shadow price = 0 indicates the constraint has slack (not binding). Adding more of that resource won't help.

Q12.3 (MCQ)

You can buy additional resource at $5/unit. The shadow price is $12. The allowable increase is 20 units. How much should you buy?

A) 0 units
B) 20 units
C) As much as possible
D) Cannot determine without more info

Answer: B (20 units)

Explanation: Since shadow price ($12) > cost ($5), buying is profitable. But shadow price is only valid within allowable range (20 units max).

Problem Set 13: Sensitivity Analysis - Allowable Ranges

Q13.1 (True/False)

The allowable increase/decrease ranges indicate how much a coefficient can change before the optimal SOLUTION changes.

☐ True
☐ False

Answer: False

Explanation: Allowable ranges indicate when the optimal BASIS changes. The optimal values of variables may change even within the allowable range, but the set of basic variables stays the same.

Q13.2 (MCQ)

A constraint RHS has allowable increase of 10 and allowable decrease of 5. Current RHS is 50. The shadow price remains valid for RHS in range:

A) [40, 55]
B) [45, 60]
C) [50, 60]
D) [40, 50]

Answer: B ([45, 60])

Explanation: Valid range = [current - decrease, current + increase] = [50 - 5, 50 + 10] = [45, 60].

Q13.3 (Fill in the blank)

If you change a RHS value beyond its allowable range, the shadow price ________ valid.

Answer: is no longer (or becomes invalid)

Explanation: Beyond the allowable range, the basis changes and you need to re-solve to find new shadow prices.

Problem Set 14: Reduced Costs

Q14.1 (MCQ)

For a NON-BASIC variable in a maximization problem, if reduced cost = -5, what should we do?

A) Current solution is optimal
B) Increase that variable to improve objective
C) Decrease that variable
D) The problem is infeasible

Answer: B

Explanation: For max problem, negative reduced cost means entering that variable improves the objective. c̄ < 0 → good to increase.

Q14.2 (True/False)

At an optimal solution for a maximization problem, all reduced costs for non-basic variables must be ≤ 0.

☐ True
☐ False

Answer: True

Explanation: This is the optimality condition for the simplex method in maximization problems. All c̄ⱼ ≤ 0 for non-basic variables.

Q14.3 (Fill in the blank)

The reduced cost for variable j is computed as c̄ⱼ = cⱼ - ________.

Answer: y'Aⱼ (or yᵀAⱼ)

Explanation: Where y is the vector of dual variables and Aⱼ is the column of constraint coefficients for variable j.

Problem Set 15: Adding Variables to an LP

Q15.1 (MCQ)

You have an optimal solution. You add a new variable to a NON-BINDING constraint. What happens to the optimal objective value?

A) Must increase
B) Must decrease
C) Stays the same
D) Cannot determine

Answer: C

Explanation: Adding a variable to a non-binding constraint won't help because that constraint isn't limiting the solution. The new variable will be 0 in the optimal solution.

Q15.2 (True/False)

Adding a new variable to a maximization LP can never make the optimal objective value worse.

☐ True
☐ False

Answer: True

Explanation: Adding variables expands the feasible region, so optimal value can only improve or stay the same (never gets worse).

Q15.3 (MCQ)

You add variable x₄ to your LP. The reduced cost c̄₄ = 3 (for max problem). What happens?

A) x₄ = 0 in new optimal solution
B) x₄ > 0 in new optimal solution
C) The problem becomes infeasible
D) Cannot determine

Answer: A

Explanation: For max problem, c̄₄ = 3 > 0 means entering x₄ would hurt the objective. So x₄ stays at 0.

Problem Set 16: Simplex Method - Basic Concepts

Q16.1 (True/False)

The Simplex method always finds the global optimum for linear programming problems.

☐ True
☐ False

Answer: True

Explanation: For LP, any local optimum is also a global optimum (due to convexity). Simplex finds a global optimum if one exists.

Q16.2 (MCQ)

The Simplex method moves from one BFS to another by:

A) Random search
B) Moving along edges of the feasible region
C) Moving through the interior of feasible region
D) Gradient descent

Answer: B

Explanation: Simplex moves along edges (1-dimensional faces) of the polyhedron, visiting vertices (extreme points/BFS).

Q16.3 (Fill in the blank)

In each iteration of the Simplex method, exactly ________ variable(s) enter the basis and ________ variable(s) leave the basis.

Answer: one (1), one (1)

Explanation: Each iteration involves one variable entering and one leaving to move to an adjacent BFS (except in degenerate cases).

Problem Set 17: Optimality Conditions

Q17.1 (MCQ)

For a MINIMIZATION problem, the optimality condition for non-basic variables is:

A) All reduced costs ≤ 0
B) All reduced costs ≥ 0
C) All reduced costs = 0
D) At least one reduced cost < 0

Answer: B (All reduced costs ≥ 0)

Explanation: For minimization, c̄ⱼ ≥ 0 for all non-basic variables means we can't improve by entering any variable. (Opposite of maximization!)

Q17.2 (True/False)

If all reduced costs are strictly positive (c̄ⱼ > 0) for a minimization problem, the optimal solution is unique.

☐ True
☐ False

Answer: True

Explanation: Strictly positive reduced costs (c̄ⱼ > 0) mean no alternative optimal solutions exist. If c̄ⱼ = 0 for some non-basic variable, there could be alternative optima.

Q17.3 (MCQ)

At a BFS, you have one non-basic variable with c̄ = 0 (all others > 0) for a min problem. This indicates:

A) Current solution is not optimal
B) Alternative optimal solutions exist
C) The problem is unbounded
D) The problem is infeasible

Answer: B

Explanation: c̄ = 0 for a non-basic variable means entering it doesn't change the objective, so there's an alternative optimal solution.

Problem Set 18: Unbounded Solutions

Q18.1 (True/False)

If an LP is unbounded, it must have at least one feasible solution.

☐ True
☐ False

Answer: True

Explanation: To be unbounded, the LP must be feasible (have at least one solution) and the objective can be improved without limit.

Q18.2 (MCQ)

In the Simplex method, unboundedness is detected when:

A) All reduced costs are negative
B) An entering variable has no valid leaving variable (all dᵢ ≥ 0)
C) The basis matrix is singular
D) All basic variables are zero

Answer: B

Explanation: If we want to increase a variable (negative reduced cost) but all direction components are ≥ 0, we can increase it indefinitely → unbounded.

Q18.3 (Fill in the blank)

If the primal problem is unbounded, the dual problem must be ________.

Answer: infeasible

Explanation: By weak duality, if primal is unbounded, dual cannot have any feasible solution (otherwise it would bound the primal).

Problem Set 19: Infeasible Problems

Q19.1 (MCQ)

Which of the following systems is infeasible?

A) x + y ≤ 5, x + y ≥ 3, x,y ≥ 0
B) x + y ≤ 5, x + y ≥ 6, x,y ≥ 0
C) x + y = 5, x - y = 1, x,y ≥ 0
D) All are feasible

Answer: B

Explanation: Cannot have x+y ≤ 5 AND x+y ≥ 6 simultaneously. These constraints are contradictory.

Q19.2 (True/False)

If both primal and dual are infeasible, this violates the strong duality theorem.

☐ True
☐ False

Answer: False

Explanation: Both primal and dual CAN be infeasible simultaneously. Strong duality only applies when at least one has a feasible solution.

Q19.3 (MCQ)

The two-phase simplex method is used to:

A) Detect unboundedness
B) Find an initial BFS
C) Compute dual variables
D) Handle integer variables

Answer: B

Explanation: Phase 1 finds an initial BFS (or determines infeasibility). Phase 2 optimizes from that starting point.

Problem Set 20: Degeneracy

Q20.1 (True/False)

A BFS is degenerate if one or more basic variables equal zero.

☐ True
☐ False

Answer: True

Explanation: Degeneracy means a basic variable = 0, implying more than n constraints are binding at that point.

Q20.2 (MCQ)

What problem can degeneracy cause in the Simplex method?

A) Infeasibility
B) Unboundedness
C) Cycling (infinite loop)
D) Non-unique optimal solution

Answer: C

Explanation: Degeneracy can cause the simplex method to cycle (visit the same BFS repeatedly) though this is rare in practice.

Q20.3 (Fill in the blank)

At a degenerate BFS, more than ________ constraints are binding.

Answer: n (where n = number of variables)

Explanation: Normally exactly n constraints are binding at a BFS. Degeneracy means > n are binding.

Problem Set 21: Reformulating Absolute Values

Q21.1 (MCQ)

To reformulate |x| ≤ 5 as linear constraints, we write:

A) x ≤ 5
B) x ≤ 5 AND x ≥ -5
C) x² ≤ 25
D) Cannot be linearized

Answer: B

Explanation: |x| ≤ 5 means -5 ≤ x ≤ 5, which gives two linear constraints: x ≤ 5 and x ≥ -5.

Q21.2 (True/False)

To reformulate the objective min |x₁| + |x₂| as a linear program, we need to introduce 2 auxiliary variables.

☐ True
☐ False

Answer: True

Explanation: Let u = |x₁| and v = |x₂|. Then min u + v with constraints: u ≥ x₁, u ≥ -x₁, v ≥ x₂, v ≥ -x₂, u,v ≥ 0.

Q21.3 (Fill in the blank)

The constraint |x - 3| ≥ 5 is equivalent to: x ≤ ________ OR x ≥ ________.

Answer: -2, 8

Explanation: |x - 3| ≥ 5 means x - 3 ≥ 5 (→ x ≥ 8) OR x - 3 ≤ -5 (→ x ≤ -2). Note: This creates a disjunction, which is harder to handle in LP!

Problem Set 22: LP Application Problems

Q22.1 (MCQ)

In a production planning problem, if the shadow price for labor hours is $15 and for machine hours is $0, which is the bottleneck?

A) Labor hours
B) Machine hours
C) Both equally
D) Neither

Answer: A (Labor hours)

Explanation: Shadow price = 0 for machines means that constraint has slack (not binding). Labor with shadow price $15 is the binding constraint (bottleneck).

Q22.2 (True/False)

In a diet problem, if a food has zero quantity in the optimal solution and positive reduced cost (for min cost), we should not include it in the diet even if it becomes cheaper.

☐ True
☐ False

Answer: False

Explanation: Positive reduced cost means the food is currently too expensive relative to its nutritional value. If it becomes cheaper, it might enter the optimal diet.

Q22.3 (MCQ)

In a transportation problem with m sources and n destinations, how many constraints are there (excluding non-negativity)?

A) m + n
B) m × n
C) m + n - 1
D) 2(m + n)

Answer: A (m + n)

Explanation: m supply constraints (one per source) + n demand constraints (one per destination) = m + n total.

Problem Set 23: Matrix Operations

Q23.1 (True/False)

For the constraint matrix A in standard form Ax = b, if A has rank m (number of rows), then we can always find a basis.

☐ True
☐ False

Answer: True

Explanation: If rank(A) = m, there exist m linearly independent columns that can form a basis matrix B.

Q23.2 (MCQ)

The basis matrix B must be:

A) Square and invertible
B) Rectangular
C) Symmetric
D) Diagonal

Answer: A

Explanation: B is m×m (square) and must be invertible (det(B) ≠ 0) to solve for basic variables: x_B = B⁻¹b.

Q23.3 (Fill in the blank)

Given basis matrix B, the basic variables are computed as x_B = ________.

Answer: B⁻¹b

Explanation: Since Bx_B = b (for basic variables), we solve to get x_B = B⁻¹b.

A) Stop making product A

If shadow price for a resource is higher than its market price, the company should buy more of that resource (within allowable range).

Problem Set 25: Advanced LP Concepts

Q25.1 (True/False)

An LP with an optimal solution must have at least one extreme point that is optimal.

☐ True
☐ False

Answer: True

Explanation: By the fundamental theorem of LP, if an optimal solution exists, at least one extreme point is optimal.

Q25.2 (MCQ)

How many extreme points can an LP have in the worst case (for a bounded polyhedron)?

A) Polynomial in n and m
B) Exponential in n or m
C) Always less than 100
D) Infinite

Answer: B

Explanation: A polyhedron defined by m constraints in ℝⁿ can have up to C(m,n) extreme points, which is exponential. E.g., hypercube has 2ⁿ vertices.

Q25.3 (Fill in the blank)

The computational complexity of the Simplex method in the worst case is ________, though it performs well in practice.

Answer: exponential

Explanation: Worst-case complexity is exponential, but average-case performance is very good (polynomial-like) for most real-world problems.

Q25.4 (True/False)

Interior point methods can solve LP in polynomial time in the worst case.

☐ True
☐ False

Answer: True

Explanation: Interior point methods (like Karmarkar's algorithm) have polynomial worst-case complexity, unlike Simplex. However, Simplex is often faster in practice.
